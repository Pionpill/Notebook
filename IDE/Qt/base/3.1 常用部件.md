    ```Qt
    void addWidget (QWidget *w)
    ```
  - addItem 加入条目
    ```Qt
    virtual void addItem (QLayoutItem *item=0 )
    ```
  - addLayout 加入子布局
    ```Qt
    void addLayout (QLayout *Layout=0 )
    ```
